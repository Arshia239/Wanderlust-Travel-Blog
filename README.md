# ✦ Wanderlust — Travel Blog

A clean, minimal travel blog built with pure HTML & CSS — no frameworks, no shortcuts. Honest travel stories from real trips, real food, and the wrong turns that made them memorable.

🔗 **Live Demo:** [wanderlust-travel-blog.netlify.app](https://wanderlust-travel-blog.netlify.app/)

## ✨ Features

- Responsive landing page with hero section
- Top Destinations showcase with cards
- Featured story / blog post section
- Photo gallery ("From the Lens")
- About section with author intro
- Fully responsive design (mobile, tablet, desktop)

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox / Grid)
